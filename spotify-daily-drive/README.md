# Spotify Daily Drive

This project creates a personalized Daily Drive-style playlist on Spotify, automatically updating every day with a mix of music and podcasts based on your preferences. It uses machine learning to fine-tune recommendations and provides a user-friendly interface for customization.

## Features

- Automatic daily playlist updates
- Machine learning-based music recommendations
- Customizable music to podcast ratio
- Ability to exclude specific artists and genres
- Dark/light mode toggle
- History tracking to avoid repeating songs too often
- Preferred time ranges for music and podcasts
- Simple caching mechanism for improved performance
- Basic analytics dashboard for user listening habits
- Comprehensive error handling and logging
